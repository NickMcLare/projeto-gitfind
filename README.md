# 🔎 GitFind

O **GitFind** é uma aplicação web desenvolvida em **React** que permite buscar repositórios do GitHub utilizando a API oficial.  
O usuário pode pesquisar um repositório específico e adicioná-lo à tela para visualizar informações e acessar o link direto para o GitHub.

---

## 🚀 Funcionalidades

- 🔍 Buscar repositórios através da API do GitHub
- 📦 Adicionar repositórios à lista
- 🔗 Acessar diretamente o repositório no GitHub
- 🚫 Evitar adicionar repositórios duplicados

## 🌐 API utilizada

Este projeto utiliza a **API pública do GitHub** para buscar informações de repositórios.

Exemplo de endpoint utilizado:

```
https://api.github.com/repos/{owner}/{repo}
```
